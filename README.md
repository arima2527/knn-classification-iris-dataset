# KNN Classification on Iris Dataset

## 📌 Project Overview
This project demonstrates the implementation of the K-Nearest Neighbors (KNN) algorithm to classify iris flowers into different species based on their features.

The Iris dataset is a well-known dataset in machine learning that contains measurements of iris flowers such as sepal length, sepal width, petal length, and petal width.

## 🚀 Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Jupyter Notebook

## 📊 Dataset
The Iris dataset contains 150 samples of iris flowers belonging to three species:
- Setosa
- Versicolor
- Virginica

Each sample has four features:
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

## 🤖 Machine Learning Algorithm

### K-Nearest Neighbors (KNN)

KNN is a supervised machine learning algorithm used for classification.  
It works by finding the **K nearest data points** to a new data point and assigning the most common class among them.

## ⚙️ Steps in the Project
1. Load the Iris dataset
2. Explore the dataset
3. Split the dataset into training and testing sets
4. Train the KNN model
5. Make predictions on test data
6. Evaluate model accuracy

## 📈 Model Evaluation
The model performance is evaluated using accuracy score and prediction results.

## 🎯 Purpose
This project demonstrates how the KNN algorithm can be used for classification problems in machine learning.
